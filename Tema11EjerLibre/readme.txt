Para el funcionamiento de este proyecto se necesitará la BBDD MySQL que se crea con el archivo .sql adjunto, 
las clases almacenadas en "src" y crear una carpeta llamada "facturas" en la raiz del proyecto.

El proyecto consiste en una tienda a la que se puede acceder tras un registro/inicio de sesión.

Una vez identificado se puede acceder a tres apartados, la tienda con el listado de los productos disponibles, 
el carro donde van apareciendo las compras y un registro de las transacciones realizadas. 

La idea original era también añadir usuarios de tipo proveedor con opciones de añadir/retirar productos y modificar
el stock de los mismos, pero por falta de tiempo este apartado no está hecho, por lo que los productos que existen 
actualmente han sido añadidos directamente a la BBDD.
